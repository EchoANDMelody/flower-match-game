# 🌸 花朵消消除 — 游戏说明

---

## 🎯 游戏目标

在有限步数内，通过交换相邻花朵使三个或更多相同花朵连成一线来消除它们，达到每关的目标分数即可过关。

---

## 🕹️ 基本操作

| 操作 | 方式 |
|------|------|
| **选中花朵** | 点击一朵花，出现粉色选中框 |
| **交换花朵** | 选中后再点击相邻花朵，或直接拖拽花朵到相邻位置 |
| **取消选中** | 点击已选中的花朵，或点击非相邻位置 |

> ⚠️ 只能交换**上下左右**相邻的花朵，不能斜向交换。交换后必须形成三消才能成功，否则会自动换回。

---

## 💥 消除规则

- **三消**：3 朵相同花朵横/竖连成一线 → 消除
- **四消**：4 朵连线 → 消除 + 额外分数
- **五消及以上**：5 朵以上连线 → 消除 + 更高分数
- **连锁反应**：消除后上方花朵下落、新花朵填充，若再次形成三消则自动连锁消除

---

## 🔥 连击系统

| 连击数 | 分数倍率 |
|--------|---------|
| 1x | 基础 10 分/朵 |
| 2x | 20 分/朵 |
| 3x | 30 分/朵 |
| ... | 每次连击 +10 分/朵 |

> 连锁消除越多，连击越高，分数越高！

## 🧪 道具系统

每关开始时会获得一定数量的道具，点击底部道具栏激活后使用。

### 💣 炸弹
- **效果**：点击棋盘任意位置，消除以该格为中心 **3×3 范围**内所有花朵
- **使用方式**：点击「💣 炸弹」→ 点击棋盘目标位置
- **音效**：💥 爆炸声
- **策略**：适合清除密集区域，一次最多消除 9 朵

### 📋 复制
- **效果**：将一朵花的类型复制到另一朵花上，使目标变成与源相同的花朵
- **使用方式**：
  1. 点击「📋 复制」
  2. 点击**源花朵**（出现紫色虚线框）
  3. 点击**目标花朵** → 目标变为源的类型
- **音效**：✨ 魔法声
- **策略**：用来制造三消，尤其当差一朵就能连线时

### 🪓 铲除
- **效果**：直接移除一朵花，该位置变为空格后上方花朵下落
- **使用方式**：点击「🪓 铲除」→ 点击要移除的花朵
- **音效**：🪓 敲击声
- **策略**：移除"挡路"的花朵，让关键花朵落到合适位置

> 💡 道具**不消耗步数**，但每关数量有限，请合理使用！

## 💡 新手技巧

1. **优先寻找四消和五消** — 一次消除越多，分数越高
2. **利用连锁反应** — 观察下落后可能产生的新匹配
3. **炸弹留到关键时刻** — 不要在简单三消就能解决时浪费炸弹
4. **复制比铲除更灵活** — 复制可以主动创造三消，铲除只是被动移除
5. **从底部消除** — 底部消除后上方花朵下落，更容易触发连锁
6. **注意步数** — 每次交换消耗 1 步，道具不消耗步数
